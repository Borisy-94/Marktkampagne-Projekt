## 📊 Customer Analytics & Classification Project
🚀 Projektübersicht

Dieses Projekt analysiert Kundenverhalten anhand von Einkommen und Ausgaben und entwickelt Machine-Learning-Modelle zur Vorhersage von Marketingreaktionen.

Ziel ist es, datenbasierte Entscheidungen im Marketing zu ermöglichen und Kundensegmente besser zu verstehen.

📌 Projektziele
Analyse von Kundenverhalten (Einkommen vs. Ausgaben)
Identifikation von Kundensegmenten
Vorhersage der Reaktion auf Marketingkampagnen
Vergleich verschiedener Machine-Learning-Modelle
📊 Explorative Datenanalyse (EDA)

Die Visualisierung zeigt verschiedene Kundensegmente:

🔵 Normale Kunden – durchschnittliches Verhalten
🟢 Luxuskunden – hohes Einkommen & hohe Ausgaben
🟡 Verdächtige Kunden – niedriges Einkommen, hohe Ausgaben
🟣 Potenzielle Sparer – hohes Einkommen, geringe Ausgaben

👉 Diese Segmentierung bildet die Grundlage für gezielte Marketingstrategien.

🧠 Methodik: Machine Learning

Zwei Modelle wurden implementiert:

1. Logistic Regression
Einfach interpretierbar
Gut für Business-Entscheidungen
2. Random Forest
Erkennt komplexe Muster
Höhere Robustheit gegenüber Datenrauschen
⚙️ Vorgehen:
Feature Engineering
Train-Test-Split (70/30)
Daten-Skalierung
Umgang mit Imbalanced Data (class_weight='balanced')
📈 Modellbewertung
Ergebnisse:
Logistic Regression: AUC = 0.89
Random Forest: AUC = 0.88

👉 Beide Modelle liefern starke Ergebnisse.

💡 Key Insights
Kundenverhalten kann klar segmentiert werden
Machine Learning ermöglicht präzise Vorhersagen
Logistic Regression ist ideal für interpretierbare Entscheidungen
Random Forest bietet zusätzliche Genauigkeit bei komplexen Mustern
